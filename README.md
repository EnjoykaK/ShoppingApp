# Einkaufsliste (Shopping List App)

Eine intuitive, grafische Desktop-Anwendung (GUI) zur Verwaltung einer Einkaufsliste. Das Projekt kombiniert **Tkinter** für die Benutzeroberfläche und **SQLite** für die dauerhafte Datenspeicherung.

---

## Projektbeschreibung

Mit dieser Anwendung können Benutzer digitale Einkaufslisten erstellen und verwalten. Alle Daten werden lokal in einer SQLite-Datenbank gespeichert, sodass die Liste auch nach dem Schließen der App erhalten bleibt.

### Hauptfunktionen
*  **Produkt hinzufügen:** Schnelle Eingabe von Produktname und Menge über Pop-up-Dialoge.
*  **Produkte anzeigen:** Live-Aktualisierung und übersichtliche Darstellung der Liste in einer Listbox.
*  **Produkt löschen:** Gezieltes Entfernen von Produkten aus der Datenbank per Namenseingabe.
*  **Dauerhafte Speicherung:** Automatische Erstellung und Nutzung einer lokalen SQLite3-Datenbank (`shopping2.db`).
*  **Visuelle Oberfläche:** Farbenfrohes GUI-Design mit individuellem App-Icon.
